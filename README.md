# Mini Pinball Game

Simple pinball-style mini game built with [Pygame](https://www.pygame.org/).

## Setup

```bash
pip install -r requirements.txt
```

## Run the Game

```bash
python pinball.py
```

## Automated Test Mode

To run the game in a headless mode for automated testing:

```bash
SDL_VIDEODRIVER=dummy python pinball.py --test
```
